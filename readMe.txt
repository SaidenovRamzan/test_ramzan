 В общем, вам нужно скачать проект (git clone ssh...). Затем установите все из reqs.txt (pip install -r reqs.txt) . 
 Запускаем celery( python -m celery -A django_celery worker) и redis (redis-server). Перейдите в папку, где вы можете найти manage.py. 
 Запустить проект(./manage.py runserver). Пароль и логин суперпользователя уже созданы, это root.
